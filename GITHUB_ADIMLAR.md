## 📋 Şimdi Yapılacaklar

### ADIM 1: GitHub'da Repository Oluştur

1. Tarayıcıda aç: https://github.com/new

2. Formu doldur:
   ```
   Repository name: gozluk-yonetim-sistemi
   Description: Profesyonel Gözlük Satış ve Stok Yönetim Sistemi
   ```

3. Seçenekler:
   - ✅ Public (herkese açık) VEYA Private (sadece sen)
   - ❌ "Add a README file" - İŞARETLEME
   - ❌ "Add .gitignore" - İŞARETLEME
   - ❌ "Choose a license" - İŞARETLEME

4. "Create repository" butonuna tıkla

### ADIM 2: Repository URL'ini Kopyala

GitHub'da oluşturduğun repository sayfasında:
- Yeşil "Code" butonuna tıkla
- HTTPS sekmesinde URL'i kopyala
- Örnek: `https://github.com/mdoganexe/gozluk-yonetim-sistemi.git`

### ADIM 3: Bana URL'i Ver

Repository URL'ini bana yapıştır, ben gerisini hallederim!

Örnek:
```
https://github.com/mdoganexe/gozluk-yonetim-sistemi.git
```

## 🔐 GitHub Kimlik Doğrulama

İlk push'ta GitHub şifre isteyecek. İki seçenek var:

### Seçenek A: Personal Access Token (Önerilen)
1. GitHub'da: Settings > Developer settings > Personal access tokens > Tokens (classic)
2. "Generate new token (classic)" butonuna tıkla
3. Note: "Gozluk Yonetim Sistemi"
4. Expiration: 90 days (veya istediğin)
5. Scope: ✅ repo (tüm repo seçenekleri)
6. "Generate token" butonuna tıkla
7. Token'ı KOPYALA (bir daha göremezsin!)
8. Push yaparken şifre yerine bu token'ı kullan

### Seçenek B: GitHub Desktop (En Kolay)
1. GitHub Desktop indir: https://desktop.github.com
2. GitHub hesabınla giriş yap
3. "Add existing repository" > Bu klasörü seç
4. "Publish repository" butonuna tıkla
